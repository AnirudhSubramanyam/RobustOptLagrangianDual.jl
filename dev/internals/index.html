RobustOptLagrangianDual.update_master_inner_level — Function
update_master_inner_level(R::Rostering, MP_inner::JuMP.Model, MP_outer::JuMP.Model, SP::JuMP.Model, master_inner::SubproblemType, λ = nothing)

Update the inner-level master model MP_inner based on the optimal values of the outer-level master model MP_outer, the subproblem model SP, and the master_inner type of the inner-level master model.
